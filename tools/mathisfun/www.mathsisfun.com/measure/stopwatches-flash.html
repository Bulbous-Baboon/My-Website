Stopwatches

Time 1, 2 or 3 things at once. Each stopwatch has Alarm mode.
You can also mark times for sports events etc. See below.


View Larger

Quick Guide

The "Play" button starts from 00:00:00.00 and the "..." button continues timing from whatever the time currently is.

The Alarm goes off at 00:00:00.00. So the time must start negative (with the "-" sign) to use the alarm.

The time/alarm can be set using the sliders for hours, minutes and seconds.

The "A", "B", etc button marks the time (without interrupting the timing)

Ideas for Timing

You can time one person, then press "+/-" and "..." , now the second person has to finish before the alarm.

Use the "A", "B", etc buttons to mark when runners finish (but keep the overall timing going).

You can start a time-handicap race by setting the three alarms with different sounds, then use the master "..." button to set them going at once.
