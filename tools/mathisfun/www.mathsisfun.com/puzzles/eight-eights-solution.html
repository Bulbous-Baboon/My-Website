Eight eights Puzzle - Solution

The Puzzle:

Use 8 exactly eight times to make 1000

a) Using only + − × /
b) Also allowing √, exponents ^, factorials !, and decimal points.

Our Solution:

A) Using only + − × /

By Shivani Dixit:
888 + 88 + 8 + 8 + 8

by Patrick Saunders:
