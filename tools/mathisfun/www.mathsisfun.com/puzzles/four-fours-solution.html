Four Fours Puzzle - Solution

The Puzzle:

A popular mathematical pastime:

Use exactly four 4's to form every integer from 0 to 50, using only the operators +, −, ×, /, () (brackets), . (decimal point), √ (square root) and ! (factorial).

Example: 0 = 44 − 44
