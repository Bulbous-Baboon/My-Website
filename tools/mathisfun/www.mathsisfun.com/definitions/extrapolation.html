Definition of

Extrapolation

more ...
Estimating a value outside a set of data points.

Example: we sell:
• 100 ice creams when it is 20°C,
• 200 ice creams when it is 25°C,
so we could extrapolate:
• 300 ice creams when it is 30°C (but we may be wrong)
