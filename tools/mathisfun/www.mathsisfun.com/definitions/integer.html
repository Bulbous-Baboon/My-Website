Integer

more ...

A number with no fractional part (no decimals).

Includes:
• the counting numbers {1, 2, 3, ...},
• zero {0},
• and the negative of the counting numbers {-1, -2, -3, ...}

We can write them all down like this: {..., -3, -2, -1, 0, 1, 2, 3, ...}

Examples of integers: -16, -3, 0, 1, 198
