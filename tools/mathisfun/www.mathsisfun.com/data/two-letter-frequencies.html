These are the 2-Letter frequencies from "Alice's Adventures in Wonderland"

Each row begins with the letter that comes first, and then we read along to find how many times each letter appears next.

Example: the first row shows you how often letters appear after an "a", so "ab" occurred 213 times, "ac" occurred 157 times, etc.

Example: run down to row "q" you will see there is only one possibility: "u" (125 times).
